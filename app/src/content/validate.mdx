# @halvaradop/ts-utility-types/validate

Helpers and type guards for runtime value validation in TypeScript and JavaScript. These functions check if a value is primitive, nullish, boolean, string, number, object, array, function, or falsy.

```ts
import * as validate from "@halvaradop/ts-utility-types/validate"
```

<h3 id="is-primitive">`isPrimitive(value: unknown): value is Primitive`</h3>

Checks if a value is a primitive type (`number`, `string`, `boolean`, `bigint`, `symbol`).

```ts
// Expected: true
isPrimitive(42)

// Expected: true
isPrimitive("hello")

// Expected: false
isPrimitive({})
```

<h3 id="is-primitive-nullish">`isPrimitiveNullish(value: unknown): value is PrimitiveNullish`</h3>

Checks if a value is a primitive or nullish (`null` or `undefined`).

```ts
// Expected: true
isPrimitiveNullish(undefined)

// Expected: true
isPrimitiveNullish("hello")

// Expected: false
isPrimitiveNullish({})
```

<h3 id="is-nullish">`isNullish(value: unknown): value is Nullish`</h3>

Checks if a value is `null` or `undefined`.

```ts
// Expected: true
isNullish(null)

// Expected: true
isNullish(undefined)

// Expected: false
isNullish(0)
```

<h3 id="is-boolean">`isBoolean(value: unknown): value is boolean`</h3>

Checks if a value is boolean.

```ts
// Expected: true
isBoolean(true)

// Expected: false
isBoolean("true")
```

<h3 id="is-strign">`isString(value: unknown): value is string`</h3>

Checks if a value is a string.

```ts
// Expected: true
isString("hello")

// Expected: false
isString(123)
```

<h3 id="is-number">`isNumber(value: unknown): value is number`</h3>

Checks if a value is numeric (including `NaN`).

```ts
// Expected: true
isNumber(42)

// Expected: false
isNumber("42")
```

<h3 id="is-object">`isObject(value: unknown): value is object`</h3>

Checks if a value is a plain object (not array, not function, not nullish).

```ts
// Expected: true
isObject({})

// Expected: false
isObject([])

// Expected: false
isObject(null)
```

<h3 id="is-array">`isArray(value: unknown): value is unknown[]`</h3>

Checks if a value is an array.

```ts
// Expected: true
isArray([1, 2, 3])

// Expected: false
isArray({})
```

<h3 id="is-function">`isFunction(value: unknown): value is Function`</h3>

Checks if a value is a function.

```ts
// Expected: true
isFunction(() => {})

// Expected: false
isFunction(123)
```

<h3 id="is-falsy">`isFalsy(value: unknown): boolean`</h3>

Checks if a value is falsy (`null`, `undefined`, `false`, `0`, `""`).

```ts
// Expected: true
isFalsy(0)

// Expected: false
isFalsy("hello")
```
