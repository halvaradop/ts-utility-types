# @halvaradop/ts-utility-types

An advanced TypeScript utility types library designed to solve complex patterns, improve type safety, and accelerate development in modern projects. It includes types for deep object manipulation, arrays, validations, testing utilities, and more.

## Installation

```bash
# Install types only for development
pnpm install -D @halvaradop/ts-utility-types@latest

# If you need the /validate entry point
pnpm install --save @halvaradop/ts-utility-types@latest
```

## Entry Points

You can import specific utilities according to the category you need:

- [`@halvaradop/ts-utility-types/deep`](/docs/deep) — deep object and type manipulation utilities
- [`@halvaradop/ts-utility-types/arrays`](/docs/arrays) — array utilities
- [`@halvaradop/ts-utility-types/objects`](/docs/objects) — object utilities
- [`@halvaradop/ts-utility-types/string-mappers`](/docs/strings) — string manipulation utilities
- [`@halvaradop/ts-utility-types/numbers`](/docs/numbers) — number utilities
- [`@halvaradop/ts-utility-types/test`](/docs/test) — type testing utilities
- [`@halvaradop/ts-utility-types/guards`](/docs/guards) — type guard utilities
- [`@halvaradop/ts-utility-types/types`](/docs/types) — deprecated entry point
- [`@halvaradop/ts-utility-types/utilities`](/docs/utilities) — deprecated entry point
- [`@halvaradop/ts-utility-types/validate`](/docs/validate) — validators and type guards
